Pubrank是一个论文投稿信息汇总平台。

## Getting Started

命令介绍

```bash
# 安装
pnpm i
# 运行
pnpm dev
# 打包
pnpm build
# 数据迁移
pnpm migrate
# prisma后台启动
pnpm admin
# 代码检验
pnpm lint
# 安装husky
pnpm prepare
# 安装husky
pnpm test
# 规范提交
pnpm commit
# 发布并生成changelog
pnpm run release -- --release-as [major or minor or patch]
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
